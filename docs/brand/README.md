# OrbitPage brand assets

The OrbitPage mark combines an open page frame with a two-node path. The shape is intentionally simple enough to stay recognizable as a browser favicon and app icon.

## Production assets

- `app/public/brand/orbitpage-mark.svg`: primary app icon and favicon.
- `app/public/brand/orbitpage-lockup.svg`: horizontal logo and wordmark.
- `app/public/brand/orbitpage-mark-512.png`: high-resolution raster app icon.
- `app/public/brand/orbitpage-apple-touch-icon.png`: 180 px Apple touch icon.
- `app/public/brand/orbitpage-favicon-32.png`: 32 px raster favicon fallback.

## Palette

- Ink: `#111827`
- Paper: `#F8FAFC`
- Orbit teal: `#20A3A8`

Keep clear space around the mark equal to at least one node width. Do not add gradients, shadows, rotations, or alternate colors inside the mark. The SVG files are the canonical production artwork.
